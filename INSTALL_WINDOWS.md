# English1000 AI Teacher - Windows 安装运行

## 1. 安装 Node.js

去 Node.js 官网安装 LTS 版本。安装完成后，重新打开 PowerShell 或命令提示符。

检查：

```powershell
node -v
npm -v
```

## 最简单方式

双击：

```text
一键安装并启动_Windows.bat
```

如果提示缺少 Node.js，就先安装 Node.js LTS，然后重新双击。

## 手动方式

### 2. 安装项目依赖

进入项目目录：

```powershell
cd English1000AI
npm install
```

## 3. 启动 App

```powershell
npx expo start
```

或者双击：

```text
start-windows.bat
```

## 4. 手机运行

1. iPhone / Android 安装 Expo Go。
2. 手机和电脑连接同一个 Wi-Fi。
3. 扫描终端显示的二维码。

## 5. 当前版本功能

- 严格按学习顺序：
  1. Dreaming English Beginner
  2. Dreaming English Intermediate
  3. Bluey Season 1
  4. Peppa Pig
  5. TED-Ed
  6. Modern Family
- 334 天、每天 3 小时计划
- 首页一键继续
- 本地保存进度
- 每日通知设置
- 精听句子循环
- 单词间隔复习
- AI 老师提示词
- 本地备份导出

## 6. 重要说明

App 不内置受版权保护的视频内容，只记录学习顺序、任务和用户自己的合法观看入口。
