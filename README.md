# English1000 AI Teacher

离线优先的 1000 小时英语学习 App。目标是让用户每天打开手机就知道学什么，减少选择成本。

## Run

```bash
npm install
npx expo start
```

然后在 iPhone 或 Android 安装 Expo Go，扫描二维码运行。

## Features

- 334 天、每天 3 小时学习路线
- 一键继续当天第一个未完成任务
- 本地保存进度
- 每 7 天复习日
- 每 100 小时检查点
- 每日通知
- 精听播放器模式
- 单词间隔复习
- AI 老师提示词
- 本地备份数据

## Notes

App 不内置 Bluey、Peppa、Disney、Modern Family 等受版权保护视频，只记录学习任务、阶段和用户自己合法观看的平台信息。
