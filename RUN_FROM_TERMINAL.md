1. Right-click the ZIP file.
2. Choose **Extract All**.
3. Open the extracted `English1000AI` folder.
4. Hold `Shift`, right-click empty space, choose **Open in Terminal**.
5. Run:

```powershell
.\RUN_ME_FIRST.cmd
```
